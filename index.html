PokeJS

BEM VINDO
Este é um projeto de estudo de JavaScript.
Após alcançar alguns marcos no PokePython (GitHub), decidi dar um tempo naquele projeto para aprender outras tecnologias.
A ideia do programa é simular uma batalha Pokémon entre dois jogadores, oferecendo uma interface semelhante aos jogos clássicos do GameBoy.
PS: A visualização em desktop ainda não está otimizada. :)
PS2: Não há interesse econômico envolvido neste projeto.
