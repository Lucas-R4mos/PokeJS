PokeJS

BEM VINDO
Este projeto foi criado para que eu possa aprender e aplicar JavaScript.
Após alcançar alguns marcos no PokePython (GitHub), decidi dar um tempo naquele projeto para aprender outras tecnologias. Agora voltei para a ideia do mesmo jogo, porém utilizando interface criada com HTML/CSS e na linguagem JavaScript.
Diferente do projeto anterior, desta vez pretendo usar uma API em vez de arquivos ".csv" para obter os dados sobre os Pokémons e sobre os golpes.
A ideia do programa é simular uma batalha Pokémon entre dois jogadores, oferecendo uma interface semelhante aos jogos clássicos do GameBoy.
Logo que surgirem funcionalidades com jogabilidade eu irei postar o projeto no Heroku :)
PS: A visualização em desktop ainda não está otimizada. Dê preferência para a visualisação em dispositivo móvel. :)
PS2: Não há interesse econômico envolvido neste projeto.
